## Cómo añadir una nueva categoría

1. Crea una carpeta con el nombre de la categoría dentro de `docs/agents/`
2. Añade un `README.md` en esa carpeta explicando el flujo de los agentes
3. Crea un archivo `.md` por cada agente
4. Registra la nueva categoría en esta tabla
